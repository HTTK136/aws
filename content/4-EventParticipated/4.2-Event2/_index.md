# Report on “Saturday Meet up - June 06, 2026”

### Purpose of the Event

+ Update participants on modern technologies in Cloud Computing, DevOps, Cyber Security, Artificial Intelligence, and Software Development.
+ Share practical experiences from industry professionals working in Cloud, System Administration, and Software Engineering.
+ Introduce AWS services such as Amazon CloudFront, AWS WAF, Amazon Bedrock, Amazon Neptune, API Gateway WebSocket, AWS Lambda, DynamoDB, and Cloud Native solutions.
+ Demonstrate how modern technologies can be applied to software development, cybersecurity, AI applications, multiplayer game development, and career growth in Information Technology.

### Speakers

+ Bao Huynh – Docker: A Containerization Technology.
+ Le Hoang Gia Dai – Machine Learning-based Network Intrusion Detection System (AWS WAF + Machine Learning).
+ Nguyen Quoc Bao – Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets.
+ Truong Huy Phuoc – The Art of Effective Teamwork.
+ Viet Phat – Build GraphRAG Applications Using Amazon Bedrock and Amazon Neptune.
+ Tran Trung Vinh – From IT Helpdesk to Senior Sysadmin.

### Key Highlights

#### Docker – A Containerization Technology

+ Introduced the concepts of Virtualization and Containerization.
+ Compared Virtual Machines and Docker Containers in terms of performance, scalability, and resource consumption.
+ Explained Docker Images, Docker Containers, and Dockerfiles.
+ Demonstrated the process of building images, creating containers, and deploying applications.
+ Presented Docker use cases in Microservices, CI/CD pipelines, Cloud Native applications, and development environments.

#### Machine Learning-based Network Intrusion Detection System

+ Introduced AWS WAF and its role in protecting web applications.
+ Explained the limitations of traditional rule-based detection against modern cyberattacks.
+ Presented a Machine Learning-based Network Intrusion Detection System (NIDS).
+ Demonstrated data preprocessing using the CSE-CIC-IDS2018 dataset, model training with LightGBM, and evaluation.
+ Showed an AWS deployment architecture using Amazon EC2, AWS Lambda, Amazon S3, Amazon CloudWatch, Amazon GuardDuty, AWS Security Hub, and Amazon SNS.
+ Demonstrated how Machine Learning enhances AWS WAF by detecting unknown attack patterns.

#### Multiplayer in the Cloud with AWS WebSockets

+ Introduced different multiplayer networking models.
+ Compared HTTP Polling, WebSocket, and UDP/ENet architectures.
+ Demonstrated a multiplayer game architecture using Amazon API Gateway WebSocket, AWS Lambda, and Amazon DynamoDB.
+ Explained player connection management, matchmaking, and real-time synchronization.
+ Demonstrated multiplayer communication using the Godot game engine.
+ Compared the WebSocket architecture with AWS GameLift for multiplayer game hosting.

#### The Art of Effective Teamwork

+ Shared four essential principles for effective teamwork.
+ Explained the importance of defining common goals and assigning the right people to the right tasks.
+ Highlighted open communication and active listening within development teams.
+ Emphasized personal accountability and collaboration.
+ Introduced productivity tools including Trello, ClickUp, Slack, Discord, and Google Workspace.

#### GraphRAG with Amazon Bedrock and Amazon Neptune

+ Introduced Retrieval-Augmented Generation (RAG).
+ Explained the limitations of traditional RAG systems.
+ Presented GraphRAG and its Multi-hop Reasoning capability.
+ Demonstrated GraphRAG implementation using Amazon Bedrock Knowledge Bases and Amazon Neptune.
+ Compared Fully Managed and Custom Pipeline architectures.
+ Showed how Knowledge Graphs improve the quality and accuracy of AI-generated responses.

#### From IT Helpdesk to Senior Sysadmin

+ Shared a real-world career journey from IT Helpdesk to Senior System Administrator.
+ Discussed essential skills for Helpdesk engineers and System Administrators.
+ Explained the transition from on-premises infrastructure to Cloud Computing and DevOps.
+ Introduced Infrastructure as Code (IaC), Docker, CI/CD, and Cloud engineering practices.
+ Shared interview experiences, portfolio development strategies, and career advice.

#### Cyber Security Knowledge

#### Artificial Intelligence Knowledge

+ Learned the principles of Retrieval-Augmented Generation (RAG).
+ Understood how GraphRAG leverages relationships between entities to improve AI responses.
+ Learned the roles of Amazon Bedrock and Amazon Neptune in building Generative AI applications.
+ Recognized the potential of AI in cybersecurity and intelligent data analysis.

#### Software Development Knowledge

+ Understood the architecture of cloud-based multiplayer game systems.
+ Learned how WebSockets enable real-time communication.
+ Understood matchmaking mechanisms and player connection management.

#### Teamwork and Career Development

+ Understood the importance of teamwork in software engineering.
+ Learned how to use collaboration and project management tools effectively.

### Practical Applications

+ Use Docker to package and deploy applications consistently across multiple environments.
+ Apply AWS WAF together with Machine Learning to strengthen cybersecurity and intrusion detection.
+ Build AI-powered applications using Amazon Bedrock and GraphRAG for intelligent knowledge retrieval.
+ Develop real-time multiplayer applications using API Gateway WebSocket, AWS Lambda, and Amazon DynamoDB.
+ Apply effective teamwork principles and collaboration tools in academic and software development projects.
+ Build a long-term learning roadmap focused on Cloud Computing, DevOps, Artificial Intelligence, and System Administration based on the speakers' practical experiences.

### Event Experience

Participating in these technical sharing sessions provided me with valuable practical knowledge in Cloud Computing, DevOps, Cyber Security, Artificial Intelligence, and Software Engineering through real-world projects and demonstrations. Each speaker not only introduced technical concepts but also shared practical challenges, implementation experiences, and lessons learned from their professional careers.

The sessions on Docker, Machine Learning integrated with AWS WAF, GraphRAG using Amazon Bedrock and Amazon Neptune, and cloud-based multiplayer game architecture helped me better understand how AWS services can be combined to build modern, scalable, and enterprise-ready solutions.

Beyond technical knowledge, the presentations on teamwork and the career journey from IT Helpdesk to Senior Sysadmin provided valuable insights into professional development, continuous learning, soft skills, and the importance of gaining practical experience through real projects. These experiences are highly beneficial for students preparing to enter the IT industry.

### Lessons Learned

+ Docker is a fundamental technology for Cloud Native application development and Microservices deployment.
+ Combining AWS WAF with Machine Learning significantly improves the detection of sophisticated cyberattacks beyond traditional rule-based methods.
+ GraphRAG extends the capabilities of Generative AI by leveraging Knowledge Graphs and Multi-hop Reasoning.
+ API Gateway WebSocket, AWS Lambda, and Amazon DynamoDB provide an effective architecture for real-time communication and multiplayer applications.
+ Strong teamwork, effective communication, and proper collaboration tools are essential for successful software projects.
+ Building a solid foundation in Linux, Networking, Cloud Computing, and DevOps, together with hands-on experience and a strong project portfolio, creates a significant advantage for long-term career growth in the Information Technology industry.
