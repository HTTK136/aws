# Bài thu hoạch “Saturday Meet up - 06/06/2026”

### Mục Đích Của Sự Kiện

+ Cập nhật kiến thức về các công nghệ hiện đại trong lĩnh vực Cloud Computing, DevOps, Cyber Security, Artificial Intelligence và Software Development.
+ Chia sẻ kinh nghiệm thực tế từ các chuyên gia đang làm việc trong lĩnh vực Cloud, System Administration và Software Engineering.
+ Giới thiệu các dịch vụ AWS như Amazon CloudFront, AWS WAF, Amazon Bedrock, Amazon Neptune, API Gateway WebSocket, AWS Lambda, DynamoDB và các giải pháp Cloud Native.
+ Hướng dẫn áp dụng các công nghệ hiện đại vào phát triển phần mềm, bảo mật hệ thống, xây dựng ứng dụng AI, Multiplayer Game và phát triển nghề nghiệp trong ngành CNTT.

### Danh Sách Diễn Giả

+ Bảo Huỳnh – Docker: A Containerization Technology.
+ Lê Hoàng Gia Đại – Machine Learning-based Network Intrusion Detection System (AWS WAF + ML).
+ Nguyễn Quốc Bảo – Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets.
+ Trương Huy Phước – The Art of Effective Teamwork.
+ Việt Phát – Build GraphRAG Applications using Amazon Bedrock and Amazon Neptune.
+ Trần Trung Vinh – From IT Helpdesk to Senior Sysadmin.

### Nội Dung Nổi Bật

#### Docker – A Containerization Technology

+ Giới thiệu khái niệm Virtualization và Containerization.
+ So sánh Virtual Machine với Docker Container về hiệu năng, khả năng mở rộng và mức tiêu thụ tài nguyên.
+ Giới thiệu Docker Image, Docker Container và Dockerfile.
+ Minh họa quy trình Build Image, tạo Container và triển khai ứng dụng.
+ Ứng dụng Docker trong Microservices, CI/CD, Cloud Native Applications và môi trường phát triển.

#### Machine Learning-based Network Intrusion Detection System

+ Giới thiệu AWS WAF và vai trò trong bảo vệ Web Application.
+ Phân tích những hạn chế của Rule-based Detection trước các cuộc tấn công hiện đại.
+ Giới thiệu Network Intrusion Detection System (NIDS) sử dụng Machine Learning.
+ Tiền xử lý bộ dữ liệu CSE-CIC-IDS2018, huấn luyện mô hình LightGBM và đánh giá kết quả.
+ Xây dựng Dashboard giám sát và kiến trúc triển khai trên AWS với EC2, Lambda, S3, CloudWatch, GuardDuty, Security Hub và SNS.
+ Kết hợp Machine Learning với AWS WAF nhằm nâng cao khả năng phát hiện các mối đe dọa mới.

#### Multiplayer in the Cloud với AWS WebSockets

+ Giới thiệu các mô hình Multiplayer Networking.
+ So sánh HTTP Polling, WebSocket và UDP/ENet.
+ Xây dựng kiến trúc Multiplayer Game sử dụng API Gateway WebSocket, AWS Lambda và DynamoDB.
+ Quản lý kết nối người chơi, Matchmaking và đồng bộ dữ liệu thời gian thực.
+ Demo kết nối nhiều người chơi bằng Godot Engine.
+ So sánh giải pháp WebSocket với AWS GameLift trong phát triển Game Server.

#### The Art of Effective Teamwork

+ Chia sẻ bốn nguyên tắc làm việc nhóm hiệu quả.
+ Xác định mục tiêu chung và phân công đúng người đúng việc.
+ Xây dựng văn hóa giao tiếp cởi mở và lắng nghe tích cực.
+ Nâng cao tinh thần trách nhiệm của từng thành viên.
+ Giới thiệu các công cụ hỗ trợ quản lý công việc như Trello, ClickUp, Slack, Discord và Google Workspace.

#### GraphRAG với Amazon Bedrock và Amazon Neptune

+ Giới thiệu Retrieval-Augmented Generation (RAG).
+ Phân tích những hạn chế của RAG truyền thống.
+ Giới thiệu GraphRAG và khả năng Multi-hop Reasoning.
+ Xây dựng hệ thống GraphRAG bằng Amazon Bedrock Knowledge Bases và Amazon Neptune.
+ So sánh kiến trúc Fully Managed và Custom Pipeline.
+ Ứng dụng Knowledge Graph để nâng cao chất lượng phản hồi của AI.

#### From IT Helpdesk to Senior Sysadmin

+ Chia sẻ hành trình phát triển nghề nghiệp từ IT Helpdesk đến Senior System Administrator.
+ Những kỹ năng cần thiết của Helpdesk và System Administrator.
+ Chuyển đổi từ hạ tầng On-Premise sang Cloud và DevOps.
+ Giới thiệu Infrastructure as Code, Docker, CI/CD và tư duy Cloud.
+ Chia sẻ kinh nghiệm phỏng vấn, xây dựng Portfolio và phát triển nghề nghiệp.

+ Biết cách lựa chọn kiến trúc phù hợp giữa WebSocket và GameLift tùy theo từng loại trò chơi.

+ Nhận thức được giá trị của việc học tập liên tục, thực hành dự án thực tế và xây dựng Portfolio.

+ Xây dựng lộ trình học tập theo định hướng Cloud, DevOps, AI và System Administration dựa trên kinh nghiệm thực tế của các diễn giả.

### Trải Nghiệm Trong Sự Kiện

Tham gia chuỗi phiên chia sẻ kỹ thuật giúp tôi tiếp cận nhiều kiến thức thực tế về Cloud Computing, DevOps, Cyber Security, Artificial Intelligence và Software Engineering thông qua những dự án đã được triển khai thực tế. Mỗi diễn giả đều chia sẻ không chỉ kiến thức chuyên môn mà còn giới thiệu các tình huống thực tế, những khó khăn gặp phải trong quá trình phát triển hệ thống cũng như các giải pháp đã được áp dụng.

Đặc biệt, các nội dung về Docker, Machine Learning kết hợp AWS WAF, GraphRAG với Amazon Bedrock và Amazon Neptune, cũng như kiến trúc Multiplayer Game trên AWS giúp tôi hiểu rõ hơn cách các dịch vụ AWS được kết hợp để xây dựng các hệ thống hiện đại, có khả năng mở rộng và đáp ứng yêu cầu thực tế của doanh nghiệp.

Bên cạnh kiến thức kỹ thuật, phần chia sẻ về Teamwork và hành trình phát triển nghề nghiệp từ IT Helpdesk đến Senior Sysadmin mang lại nhiều bài học về kỹ năng mềm, tư duy học tập, xây dựng lộ trình nghề nghiệp và tầm quan trọng của việc thực hành thông qua các dự án thực tế. Đây là những kinh nghiệm rất hữu ích đối với sinh viên đang chuẩn bị bước vào môi trường làm việc chuyên nghiệp.
