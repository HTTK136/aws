### Results Achieved in Week 1:

* Gained a fundamental understanding of Cloud Computing concepts and the differences between traditional on-premises infrastructure and cloud-based infrastructure.

* Learned about AWS Global Infrastructure, including Regions, Availability Zones, and Edge Locations, and understood their roles in building highly available and fault-tolerant systems.
* Successfully created an AWS Free Tier account and became familiar with the AWS Management Console interface.
* Installed and configured AWS CLI on a local machine, enabling command-line interaction with AWS services.
* Performed basic AWS CLI commands to retrieve information, configure credentials, and manage AWS resources.
